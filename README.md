# FinArc

Simple USDC payment links for freelancers.

---

## Overview

FinArc is a lightweight payment tool concept built on Arc.

The goal is straightforward:

- Create a payment link  
- Send it to a client  
- Get paid in USDC  

No banks. No friction. Just a simple payment flow.
